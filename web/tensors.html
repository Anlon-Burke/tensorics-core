Tensors

Keywords:

  • concept of pair Position and Value. Where Position is an encapsulated set of Coordinate that unambiguously points to one Value.
  • each Tensor has a Shape, that decribes all possible Positions in it.
  • On creation user have to define super type of your coordinates and type of values will be used.
  • when you build your tensor, all coordinates (a full Position acordingly to creation constraint) have to be provided while adding a Value
